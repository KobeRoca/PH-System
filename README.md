# PhilHealth System

A web-based PhilHealth Management System developed using PHP, MySQL, and XAMPP.  
It provides secure authentication, employee management, history tracking, and full CRUD operations for efficient administrative record handling.

---

## 🚀 Features

- Secure login authentication
- Employee management (Add, Update, Delete, View)
- Data history tracking
- Search and filter records
- Bulk delete (by date / all records)
- Print functionality
- Admin dashboard
- File upload management

---

## 🛠️ Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP (Local Server)

---

---

## ⚙️ How to Run the Project

1. Install XAMPP  
2. Copy project folder to:C:\xampp\htdocs\
3. Start **Apache** and **MySQL** in XAMPP  
4. Import the database using phpMyAdmin  
5. Open browser and go to:http://localhost/PH/
